# DuckGame

## Concept
> 오리게임은 고전 게임 덕헌트에서 영감을 받아 웹으로 제작하게 되었다.
 
## UIUX
>
## Click
> 클릭을 통해 오리를 사냥하는 게임이다.

## Clock -> Count
> 특정 시간이 되었을 때 오리가 생성된다. 그리고 특정 시간이 지나면 오리는 사라진다.
